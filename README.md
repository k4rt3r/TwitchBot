# TwitchBot

Send messages to Twitch.

### Functions
Currently capable of:
- Playing a song from Pretzel
- Giveaway winner picker
- Random greetings from greetings set
- Twitter spam
- Much more to come

### Develop

```bash
npm run dev
```

### Use

- Configure your environment variables
- Set your greetings ands commands

```bash
npm run start
```
